# ici01_PublicOpinionAnalyzer

## Python version

This project targets Python 3.12. Please use Python 3.12 for development and deployment (we declare
`requires-python = ">=3.12,<4.0"` in `pyproject.toml`). If you use pyenv, you can set it locally with:

```
pyenv local 3.12.0
```

Also included: `.python-version` (for pyenv) and `runtime.txt` (Heroku-style) both set to 3.12.0.
